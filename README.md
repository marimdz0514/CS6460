# CS6460: Content Project - EdTech

This is the project repository for my CS6460 Educational Technology content project.  
It is a web application designed to help students learn Python programming through interactive lessons, quizzes, and projects. The app provides a structured curriculum divided into multiple units, making it easier for beginners to build their skills step-by-step.

# Running the Application Locally

To run the application locally:

1. Open your terminal.
2. Navigate into the project directory:
   ```bash
   cd my-app
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and go to:
   ```
   http://localhost:5173/
   ```
